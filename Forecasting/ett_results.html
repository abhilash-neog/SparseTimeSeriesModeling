Training Procedure:

  • Data was split in the ratio = 6:2:2 (standard for ETT)
  • Lookback Window = 336 timesteps
  • Horizon Window = 96 timesteps
  • Pre-trained epochs = 30
  • Fine-tune epochs = 50
  • 3 Models were pretrained on 60% of each dataset with following mask ratios: 25, 50, 75
  • Once pretrained, each model was used to finetune (or applied directly) on each dataset (60% portion of it) and tested on the 20%
